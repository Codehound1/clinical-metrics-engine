# Clinical Metrics Engine (CME)

The Clinical Metrics Engine is a vendor-agnostic, SQL-first analytics framework
for building clinical, operational, and regulatory metrics across healthcare systems.

## What this repo demonstrates
- Event-driven clinical data modeling
- Time-zero logic (e.g., Sepsis)
- Metric standardization (Sepsis, CHG, ED Flow, Medication Estimates)
- EHR-agnostic fact/dimension patterns
- SQL Server and PostgreSQL compatibility

## Use cases
- Quality & regulatory reporting
- Clinical operations dashboards
- Analytics engineering patterns for healthcare
- Education and portfolio demonstration

This repository accompanies the book:
**Healthcare Data Engineering & Analytics: A Practical Guide**
